# Bruno Firmo
### Full Stack Developer · SaaS · IA aplicada · Sistemas financeiros

Desenvolvedor full stack focado em <b>plataformas SaaS multi-tenant</b>, <b>módulos financeiros</b> e <b>integrações de IA</b> que resolvem problema de negócio de verdade, do conceito à produção.

Trabalho principalmente com o ecossistema <b>.NET</b>, <b>React / React Native</b>, <b>Python</b> e <b>SQL Server / PostgreSQL</b>, com experiência em sistemas corporativos de larga escala, design de APIs, modelagem de dados, regras de negócio complexas e arquitetura de sistemas.

Atualmente me especializo em <b>agentes de IA confiáveis</b>: arquiteturas onde o LLM interpreta e narra, tudo isso com cálculo determinístico, tool use e validação de proveniência. É a filosofia por trás dos produtos financeiros que construo.
